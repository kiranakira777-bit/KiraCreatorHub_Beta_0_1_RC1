# PROJECT RULE - KIRA CREATOR HUB

1. Jangan mengubah struktur project tanpa alasan kuat.
2. Semua fitur baru dibuat sebagai modul baru di dalam `assets/js/modules/`.
3. Jangan menghapus fitur lama.
4. Semua update harus backward compatible.
5. Semua data dipisahkan dari UI (simpan di folder `data/`).
6. Semua konfigurasi dipisahkan dari logic (simpan di `config/`).
7. Semua teks menggunakan Bahasa Indonesia yang sederhana.
8. Semua icon menggunakan Lucide Icons (SVG).
9. Jangan menggunakan emoji pada UI.
10. Selalu dokumentasikan perubahan pada CHANGELOG.md.
